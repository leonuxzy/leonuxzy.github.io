Vote的快捷键

发表于 2021-06-12 更新于 2022-01-03 分类于 工具说明 阅读次数：
本文字数： 457 阅读时长 ≈ 1 分钟

本文将用列表的方式，简单介绍笔记软件Vnote的主要操作。

1. 软件快捷键

快捷键 操作
Ctrl + Alt + S 新的子文件夹
Ctrl + 8 下一个匹配
Ctrl + 9 之前的 Match
Ctrl + F 找到
Ctrl + Alt + N 新的 Note
Ctrl + T 编辑阅读 Note
Ctrl + Tab 激活下一个选项卡
F11 全屏
Ctrl + S 保存 Note
Ctrl + Alt + I 快速访问
Ctrl + Shift + Tab 激活之前的标签
Shift + F3 找到以前的
Ctrl + Alt + F 先进的发现
Ctrl + E 队长模式
Ctrl+Shift+T 最后关闭的文件
F12 打开通过默认程序
Ctrl + Alt + L Flash页面
Ctrl+Shift+V 粘贴纯文本
Ctrl + G 通用输入
F3 查找下一个

2. 编辑快捷键

快捷键 操作
Ctrl+B 插入粗体
Ctrl+I 插入斜体
Ctrl+D 插入删除线
Ctrl+; 插入行内代码
Ctrl+M 插入代码块
Ctrl+L 插入链接
Ctrl+. 插入表格
Ctrl+’ 插入图片
Ctrl+ 插入级别为的标题
Ctrl+7 删除当前行或所选择文本的标题标记

在此，将测试跨页锚点的功能，测试顺利！
